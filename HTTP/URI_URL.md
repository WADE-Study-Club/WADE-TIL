# URI/URL

### URI(Uniform Resource Identifiers)

: 스키마(http: 와 ftp 등)를 나타내는 `리소스를 식별`하기 위한 식별자입니다.

### URL(Uniform Resource Locator)

: `리소스의 장소`(네트워크 상의 위치)를 나타냅니다.

- URL는 RUI의 하위집합 입니다.
- 이해하기 편하게 URI를 URL로 변경하여도 아무런 지장이 없습니다.
- URL 포맷에는 절대 URL, 상대 URL이 있습니다.
